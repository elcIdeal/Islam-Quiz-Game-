Game Rules

Objective:
Answer all quiz questions correctly by choosing the right option from four choices.

Question Structure:
Each question has four answer options, with only one correct answer. Choose the answer you think is correct.

Timer:
You have 10 seconds to answer each question. If time runs out, the question will be marked as incorrect.

Music:
Background music can be toggled on or off using the sound button at the top. Music is optional and doesn’t affect gameplay.

Navigation:
After selecting an answer, the game will automatically move to the next question.
